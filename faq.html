C'est quoi une skill (recette) ?
What is a skill?
Un ensemble de procédés : du code, des requêtes, des règles de présentation, des bonnes pratiques. Pour un agent IA ou humain.
A set of processes: code, queries, presentation rules, best practices. For an AI or human agent.
C'est quoi une hyperskill (recette) ?
What is a hyperskill?
Une skill encodée en base64 dans le paramètre hs d'une URL. Comme une page web est un document accessible via une URL, une skill est un fichier d'instruction accessible via une hyperskill. Voir hyperskills.net.
A skill base64-encoded in the hs parameter of a URL. As a web page is a document accessible via a URL, a skill is an instruction file accessible via a hyperskill. See hyperskills.net.
Pourquoi un paramètre hs ?
Why an hs parameter?
Le paramètre hs rend la skill lisible, bookmarkable et transmissible directement à un agent IA. La skill voyage avec l'URL.
The hs parameter makes the skill readable, bookmarkable, and directly transmittable to an AI agent. The skill travels with the URL.
Quelle taille maximum ?
Maximum size?
Dépend du navigateur : Chrome/Edge ~32 000 caractères, Firefox ~65 000, Safari ~80 000. Avec compression (préfixe gz. ou br.), des skills de 250-500 Ko tiennent dans ~32 000 caractères.
Depends on the browser: Chrome/Edge ~32,000 characters, Firefox ~65,000, Safari ~80,000. With compression (prefix gz. or br.), skills of 250-500 KB fit in ~32,000 characters.
Quel format de contenu ?
What content format?
Libre. Markdown, SQL, YAML, HTML, Svelte, texte. Le format Agent Skills est recommandé mais pas obligatoire.
Free. Markdown, SQL, YAML, HTML, Svelte, text. The Agent Skills format is recommended but not required.
Quelle licence ?
What license?
Les skills sur Skillpedia sont sous licence creative commons.
Skills on Skillpedia are under creative commons license.
Versioning ?
Versioning?
Le hash de l'URI source + contenu sert de trace. Chaque version peut référencer le hash de la version précédente. Recommandé, pas obligatoire.
The hash of source URI + content serves as a trace. Each version can reference the previous version's hash. Recommended, not required.
Spam ?
Spam?
Le format hyperskills est ouvert. Skillpedia, en tant que répertoire, impose ses propres règles pour l'indexation (signature, identité, etc.).
The hyperskills format is open. Skillpedia, as a directory, imposes its own rules for indexing (signature, identity, etc.).
MCP ?
MCP?
MCP connecte les agents IA aux données. Une skill peut contenir des requêtes vers un serveur MCP.
MCP connects AI agents to data. A skill can contain queries to an MCP server.
WebMCP ?
WebMCP?
WebMCP expose des outils dans le navigateur. Une skill peut utiliser ces outils.
WebMCP exposes tools in the browser. A skill can use these tools.
Agent Skills ?
Agent Skills?
Agent Skills est un format ouvert pour les skills (SKILL.md). Skillpedia accepte ce format mais ne l'impose pas.
Agent Skills is an open format for skills (SKILL.md). Skillpedia accepts this format but does not require it.
Open-Root ?
Open-Root?
Open-Root, créé par Louis Pouzin (inventeur du datagramme). Système de nommage internet indépendant d'ICANN.
Open-Root, created by Louis Pouzin (inventor of the datagram). Internet naming system independent of ICANN.
