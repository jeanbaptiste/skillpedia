## Context

In 1990, Tim Berners-Lee wrote WorldWideWeb on a NeXT at CERN.
It was both a browser and a page editor. In 1993,
Mosaic popularized the browser without the editor.

In 1973, Louis Pouzin invented the datagram, a network packet
that carries its own routing. In 2010, he created Open-Root, an
internet naming system independent of ICANN.

## Skills

A skill is an instruction file: code, queries,
presentation rules, best practices. It helps an
agent (AI or human) accomplish a task.

Anthropic formalized skills for Claude. OpenAI uses them
in Codex. GitHub Copilot, Cursor, VS Code support them. The
Agent Skills format (agentskills.io) is an open standard.

## Hyperskills

A hyperskill (hypertext skill) is a skill base64-encoded
in the fragment of a URL. The skill content travels with
the link. See hyperskills.net.

By analogy:
- a web page is a document accessible via a URL
- a skill is an instruction file accessible via a hyperskill

## Skillpedia

Skillpedia (skillpedia.eu) is a directory of hyperskills.

Skills on Skillpedia are commons under Creative Commons license.

The content format is free: Markdown, SQL, YAML, HTML, Svelte,
plain text. The Agent Skills format is recommended but not required.

Hyperskills can also be hosted elsewhere: on IPFS,
on Open-Root, on a personal server. Skillpedia is a
directory.
