## Contexte

En 1990, Tim Berners-Lee écrit WorldWideWeb sur un NeXT au CERN.
C'est à la fois un navigateur et un éditeur de pages. En 1993,
Mosaic popularise le navigateur sans éditeur.

En 1973, Louis Pouzin invente le datagramme, un paquet réseau
qui porte son propre routage. En 2010, il crée Open-Root, un
système de nommage internet indépendant d'ICANN.

## Skills

Une skill est un fichier d'instruction : du code, des requêtes,
des règles de présentation, des bonnes pratiques. Elle aide un
agent (IA ou humain) à accomplir une tâche.

Anthropic a formalisé les skills pour Claude. OpenAI les utilise
dans Codex. GitHub Copilot, Cursor, VS Code les supportent. Le
format Agent Skills (agentskills.io) est un standard ouvert.

## Hyperskills

Une hyperskill (hypertext skill) est une skill encodée en base64
dans le fragment d'une URL. Le contenu de la skill voyage avec
le lien. Voir hyperskills.net.

Par analogie :
- une page web est un document accessible via une URL
- une skill est un fichier d'instruction accessible via une hyperskill

## Skillpedia

Skillpedia (skillpedia.eu) est un répertoire d'hyperskills.

Les skills sur Skillpedia sont des communs sous licence creative
commons.

Le format du contenu est libre : Markdown, SQL, YAML, HTML, Svelte,
texte libre. Le format Agent Skills est recommandé mais pas obligatoire.

Les hyperskills peuvent aussi être hébergées ailleurs : sur IPFS,
sur Open-Root, sur un serveur personnel. Skillpedia est un
répertoire.
